# Simple-Python-Scraper-Example-Scrape-Book-Reviews-from-Goodreads
Scrape book reviews from Goodreads, 

used package:
* `bs4` 
* `requests`
* `selenium`
* `pandas`

including features of:
* name: reviewer name;
* rating: book rating;
* date: review create date;
* review: user review content;
* like: the likes of this review;
